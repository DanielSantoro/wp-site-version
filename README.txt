A quick and easy tool that allows you to add text and styling to the admin bar so you can tell which version of the site you're working on.

== Description ==

Created for those who develop a site with live, staging, and development copies and struggle to quickly eyeball which version of the site they are on, this plugin allows you to add text and styling to the WordPress Admin Bar (at the top of all pages when logged in as an administrator) so you never make a dev change on a live site again.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `wp-site-version.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
